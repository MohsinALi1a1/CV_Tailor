# CV Tailor — Screenshots & Demo Assets

This folder hosts media that is referenced from the project README.

## How to add screenshots

Capture the following views (PNG, ≤ 1 MB each, ~1600 px wide) and drop them
in this folder using the **exact filenames** below — the README links will
resolve automatically:

| Filename | What to capture |
| --- | --- |
| `01-hero.png` | The landing view: sidebar + hero banner + all 10 tabs visible |
| `02-build-tab.png` | The **Build CV** tab with a parsed resume preview |
| `03-ats-tab.png` | The **ATS Score** tab showing the score ring + 16+ checks |
| `04-tailor-tab.png` | The **Tailor** tab after running against a sample JD |
| `05-cover-letter-tab.png` | The **Cover Letter** tab with a generated letter |
| `06-templates.png` | Side-by-side of the 3 PDF templates (minimal / professional / modern) |
| `07-cli.png` | A terminal showing `python main.py ats --resume ...` output |

Suggested capture tools:
- Windows: `Win + Shift + S` (Snipping Tool)
- macOS: `Cmd + Shift + 4`
- Cross-platform: [ShareX](https://getsharex.com/) or
  [Flameshot](https://flameshot.org/)

## How to add a demo video

1. Record a 60–90 second walkthrough (upload resume → ATS → tailor → download).
2. Upload it to YouTube as **Unlisted** (or to Loom).
3. Edit the top of `README.md` and replace the
   `https://www.youtube.com/watch?v=DEMO_VIDEO_ID`
   placeholder with the real URL.

If you'd rather embed an inline GIF instead, drop a file named
`demo.gif` (≤ 5 MB) into this folder — the README already references it.
